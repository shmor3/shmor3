I'm a systems engineer drawn to the hard, unglamorous layers — runtimes,
build systems, and the plumbing that makes AI agents actually work.

Lately that means designing a plugin-based **agent runtime in Rust**. But
I'm just as happy debugging firmware, pushing bytes across an FPGA's DMA,
or squeezing a build pipeline until it's fast.

Rust-first, pragmatic everywhere else. Always closer to the metal than I
strictly need to be.

<br clear="right" />

### What I'm into

🤖 &nbsp;**Agentic systems** — plugin runtimes, MCP / ACP tooling, local-first LLM apps
⚙️ &nbsp;**Close to the metal** — Rust internals, embedded firmware, FPGA / DMA, a little CUDA
🏗️ &nbsp;**Build & distributed systems** — concurrent build tooling and worker orchestration
